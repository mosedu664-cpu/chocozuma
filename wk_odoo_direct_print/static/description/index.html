Frequently Asked Questions

Is this a subscription-based module or a one-time purchase?

Odoo Print Direct is a one-time purchase module. There are no hidden charges and no subscription fees. You only pay once for the module.

Can I use this module to print POS receipts automatically?

To enable seamless printing in POS, we provide a dedicated module called POS Print Direct, which is an extension of the Print Direct module. This POS module allows automatic printing of POS receipts after order validation.

Can I print remotely if my host machine and Odoo users are on different devices?

Yes, remote printing is supported. You just need to ensure that the desktop/mobile application (client-side application) is installed on the host machine where the printers are connected. Odoo users can trigger printing from any device, and the print job will be sent to the host machine.

What types of printers are supported by this module?

Our module supports a wide range of printers, including ZPL, ESC/POS, and network (WiFi/Bluetooth) printers. If you would like to test your printer, you can create a support ticket at support@webkul.com, and we will be happy to share a demo with you.

Does the module work on the Mac operating system also?

Yes, the desktop application is fully compatible with macOS and has been tested on macOS Sequoia. If you have any additional questions or need assistance, feel free to contact our support team.

Can I install the host machine on an Android device or an iOS device?

Yes, our mobile application is available for Android and iOS on their respective app stores. You can install the application on these devices and use them as host machines for printing.

Can we configure multiple printers per user or per company?

Yes, you can configure printers based on users and companies. This allows you to assign different printers to different users or companies as per your business workflow.

Can I automate the printing of the delivery slip when a sales order is validated?

Yes, you can automate printing from the configuration settings. Once configured, the delivery slip will be printed automatically when the relevant document is validated.
